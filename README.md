# 智能停车系统 python703

## 项目概述

智能停车系统是一款基于Python的Django框架开发的应用程序，旨在通过数字化管理提高停车效率，实现车位信息的实时更新与管理。

## 技术栈

- **后端框架：** Django
- **编程语言：** Python
- **数据库：** MySQL

## 功能模块

### 管理员功能模块

- **车辆信息管理：**
  - 查看车主信息
  - 查看和修改车辆信息
  - 管理车位类型与信息
- **车辆出入管理：**
  - 车辆入场记录
  - 车辆出场记录
- **系统管理：**
  - 个人中心
  - 数据查看、修改与删除
  - 新增信息

### 车主功能模块

- **账户管理：**
- **信息查询：**
  - 查看首页
  - 查看车位信息
  - 查看公告信息
  - 个人中心详情查询

## 系统角色

- 管理员：负责系统管理和车辆信息维护
- 车主：进行账户和个人信息查询

## 运行环境

- Python环境搭建（推荐使用虚拟环境）
- MySQL数据库安装与配置
- Django框架部署

## 部署步骤

1. 配置Python环境
2. 安装MySQL并创建数据库
3. 克隆或导入项目源码
4. 安装依赖包
5. 数据库迁移
6. 运行Django项目
